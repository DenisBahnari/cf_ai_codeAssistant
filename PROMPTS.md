# My Most Important Prompts for LLMs

This section contains the most important questions and prompts I  asked an LLM for this project.

---

**Prompt:**

How does the Cloudflare dashboard platform work and how should I start developing a project from 0?  
What are the concepts and tools that I need to know to build an AI-powered application?

---

**Prompt:**

How can I use Cloudflare Durable Objects to persist information?  

---

**Prompt:**

What is the ideal way to ask an AI agent to perform an action inside code using natural language?  

---

**Prompt:**

How can I dynamically change an AI agent system prompt?  
Right now, the system value is fixed at the moment the agent object is created, how can I update it later?

---

**Prompt:**

Give me a basic HTML template that allows me to send questions and
display agent answers

---

**Prompt:**

This is the streaming documentation, help me implement streaming responses for my agent and to save the final generated text to the chat history once streaming is complete.

---

**Prompt:**

What is the best way to implement speech to text in web, how can I make it always listen and only trigger after for example 3 seconds of silence?

---

**Prompt:**

How does Cloudflare D1 work, and how do I add it to my project with a schema of the tables?  

---


**Prompt:**

How does SQLite syntax work?

---

**Prompt:**

How can I get the content of files from the frontend?  

---

**Prompt:**

How can I store project context in the frontend cache?  

---


**Prompt:**

I have a js file that dynamically generates HTML.  
I want you to only upgrade the HTML and CSS to make it more beautiful and responsive, a modern looking aproach with white and orange theme.

---

**Prompt:**

What is a good way to filter and detect code blocks in streaming responses?  
For example, handling content between triple backticks like ``` <code> ```

---

I also used LLMs to help me fix and understande some bugs and to help me guarantee the agents have good prompts to do what they are designed to do without alucinations.
